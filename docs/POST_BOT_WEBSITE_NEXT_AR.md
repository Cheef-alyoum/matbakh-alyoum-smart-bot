# ما بعد البوت والموقع

## Meta
1. ربط Webhook النهائي
2. تفعيل الرسائل والقوالب
3. تعبئة Pixel / CAPI
4. تعبئة صور وروابط المنتجات في الكتالوج
5. اختبار Lead / QualifiedLead / Purchase

## Google
1. ربط الدومين الرسمي
2. التأكد من HTTPS
3. إرسال Sitemap
4. فحص Coverage
5. ربط Google Business Profile بالموقع
6. مراجعة Titles / Meta / JSON-LD

## التشغيل الداخلي
1. تدريب الإدارة على:
   - /pending
   - /view ORDER_ID
   - /approve ORDER_ID
   - /reject ORDER_ID سبب
   - /ready ORDER_ID
   - /out ORDER_ID
   - /delivered ORDER_ID
2. مراجعة المناطق وأسعار التوصيل كلما تغيّرت.
3. تحديث المنيو عند أي تغيير سعر أو حالة.
4. أخذ نسخة احتياطية من Supabase قبل أي تغيير كبير.
