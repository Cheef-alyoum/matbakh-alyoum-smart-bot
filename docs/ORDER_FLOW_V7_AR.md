# وثيقة تدفق الطلب النهائي v7

## رحلة العميل
1. ترحيب باللغة
2. الموافقة على رسائل الخدمة
3. القائمة الرئيسية
4. اختيار القسم الرئيسي
5. التفرعات الخاصة بالقسم (نوع اللحم / حالة الصنف / الصنف / الكمية / الإضافات)
6. مراجعة السلة
7. اليوم
8. الساعة
9. طريقة الاستلام
10. القطاع/المحافظة
11. المنطقة
12. العنوان أو الموقع
13. طريقة الدفع
14. الملاحظات
15. الملخص النهائي للعميل
16. الإرسال إلى الإدارة
17. إشعارات الحالة حتى التسليم

## رحلة الإدارة
- يصل الطلب مع ملخص كامل
- أزرار الإدارة: موافقة / تعديل / رفض
- بعد الموافقة: تحضير / جاهز / توصيل
- عند التوصيل: تم التسليم

## منطق منع الطلب الجديد
- طالما أن هناك طلبًا مفتوحًا أو طلبًا بانتظار اعتماد الإدارة، يمنع النظام بدء طلب جديد
- أي رسالة من العميل خلال هذه الفترة تعتبر متابعة أو تتبعًا للطلب الحالي

## أرقام الطلبات
- النظام يصدر أرقامًا داخلية من النمط MAE001, MAE002, ...
- رقم الهاتف هو المرجع الأساسي للتتبع وخدمة العملاء
