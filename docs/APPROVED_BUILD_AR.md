# وثيقة البناء المعتمدة

هذه الحزمة مبنية على:
1. ملف `قائمة_مطبخ_اليوم_للبرمجة_وميتا.xlsx`
2. التعديلات التشغيلية المعتمدة داخل المشروع
3. متطلبات الظهور الرسمي والثقة وSEO وMeta وCAPI

## قواعد تشغيلية أساسية
- كل طلب جديد يدخل بحالة `قيد المراجعة`.
- لا يتم التأكيد النهائي إلا من الإدارة المخولة.
- الواجهة العامة للموقع بشرية وقابلة للفهرسة.
- الـ JSON التشغيلي متاح على `/api/status` وليس على جذر الموقع.
- دعم Lead وInitiateCheckout وQualifiedLead وPurchase في Meta CAPI.
- دعم Webhook لواتساب ورسائل نصية وصوتية.
- تصنيف العملاء حسب الموافقات التسويقية يبقى منطقًا معتمدًا في التطوير القادم.

## ما تم تضمينه
- موقع عام احترافي
- منيو عام قابل للبحث
- صفحة طلب
- صفحة تتبع
- صفحة ثقة واعتماد رقمي
- ملفات SQL
- ملفات بيانات JSON/CSV
- Webhook واتساب
- Meta CAPI
- SEO / OG / Schema / robots / sitemap

## عناصر ما زالت تحتاج تعبئة تشغيلية
- ADMIN_NUMBERS
