# تحديث v3 — أزرار واتساب + Supabase

## ما الذي تغير
- تم تحويل الردود الأولى من نص عادي إلى أزرار WhatsApp تفاعلية.
- تم إضافة رسالة موافقة بعد اختيار اللغة.
- تم إضافة القائمة الرئيسية بأزرار: ابدأ الطلب / المنيو / موظف مباشر.
- تم إضافة قائمة تفاعلية لأقسام المنيو.
- تم حفظ جلسة العميل وحالة الموافقة واللغة داخل Supabase عند توفر الإعدادات.
- عند عدم توفر Supabase يستمر النظام بالعمل على تخزين JSON المحلي.

## متغيرات البيئة المطلوبة
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- SUPABASE_SCHEMA=public
- WHATSAPP_VERIFY_TOKEN
- WHATSAPP_PHONE_NUMBER_ID
- WHATSAPP_ACCESS_TOKEN
- WHATSAPP_HUMAN_ESCALATION_NUMBER
- BASE_URL

## الجداول المطلوبة في Supabase
نفّذ الملف:
- database/schema.sql
- database/seed_menu.sql

## المسار الصحيح لويبهوك واتساب
- /api/webhooks/whatsapp

## السلوك الحالي
1. أول رسالة من العميل → أزرار ترحيب
2. اختيار اللغة → أزرار الموافقة
3. الموافقة → أزرار القائمة الرئيسية
4. الضغط على المنيو → قائمة أقسام تفاعلية
5. الضغط على قسم → رسالة أسعار مختصرة + رابط الطلب

## ملاحظة تشغيلية
إذا كانت الروابط في الرسائل تشير إلى الدومين الرسمي قبل ربطه، غيّر BASE_URL مؤقتًا إلى رابط onrender.
