# مطبخ اليوم المركزي — رحلة واتساب النهائية v1.3.0

## رحلة العميل
1. رسالة ترحيب مع أزرار:
   - العربية
   - English
   - موظف مباشر
2. رسالة موافقة:
   - أوافق
   - خدمة فقط
   - لا أوافق
3. القائمة الرئيسية:
   - ابدأ الطلب
   - المنيو
   - موظف مباشر
4. اختيار القسم
5. اختيار الصنف
6. اختيار الكمية
7. مراجعة السلة
8. التوصيل أو الاستلام
9. اختيار الوقت
10. إرسال العنوان أو الموقع
11. اختيار الدفع
12. إضافة ملاحظات أو تجاوزها
13. إنشاء الطلب بحالة `قيد المراجعة`
14. إشعار الإدارة مباشرة

## أوامر الإدارة في واتساب
- `/pending`
- `/view ORDER_ID`
- `/approve ORDER_ID`
- `/reject ORDER_ID سبب`
- `/ready ORDER_ID`
- `/out ORDER_ID`
- `/delivered ORDER_ID`
- `/help`

## متغيرات البيئة المهمة
- `WHATSAPP_VERIFY_TOKEN`
- `WHATSAPP_PHONE_NUMBER_ID`
- `WHATSAPP_ACCESS_TOKEN`
- `ADMIN_NUMBERS`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `BASE_URL`

## قاعدة البيانات
شغّل:
1. `database/reset_schema_v4.sql`
2. `database/seed_menu.sql`

## ملاحظات تشغيلية
- كل طلب جديد يدخل أولًا بحالة `قيد المراجعة`
- لا يتم اعتماد أي طلب نهائيًا إلا من الإدارة
- إذا لم تكن Supabase مفعلة، يعمل النظام على التخزين المحلي داخل `storage/`
