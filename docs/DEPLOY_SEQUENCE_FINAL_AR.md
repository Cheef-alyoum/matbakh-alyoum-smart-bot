# تسلسل النشر النهائي

1. نفّذ `database/reset_schema_v4.sql` داخل Supabase.
2. نفّذ `database/seed_menu.sql` داخل Supabase.
3. اضبط متغيرات البيئة في Render.
4. ارفع المشروع إلى GitHub.
5. نفّذ Deploy على Render.
6. اختبر الموقع: `/`, `/menu.html`, `/order.html`, `/track.html`, `/catering.html`, `/ready-to-cook.html`, `/frozen-chilled.html`.
7. اختبر البوت من أول رسالة حتى تثبيت الطلب والتحديثات اللاحقة.
8. بعد نجاح ذلك انتقل إلى ربط Meta وGoogle.
