# سيناريو واتساب المعتمد في هذه الحزمة

## المسارات الحالية
- منيو / أسعار
- طلب
- تتبع
- تحويل لموظف
- رسائل صوتية

## الردود الأساسية
- `منيو` => رابط المنيو
- `طلب` => رابط صفحة الطلب
- `تتبع` => رابط صفحة التتبع
- `موظف` أو `اتصال` => رقم الموظف المباشر
- رسالة صوتية => تأكيد الاستلام وتحويلها للمتابعة

## ملاحظة
منطق الموافقات التفصيلي (اللغة > الإخلاء > الموافقة > التصنيف) معتمد وظيفيًا، لكنه يحتاج ربطًا أعمق بقاعدة البيانات وحالة كل عميل عند التوسعة التالية.
